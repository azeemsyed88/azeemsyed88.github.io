    Research: Cloud Application Security Testing in Azure at USF

    Detailed research on testing cloud hosted applications in Azure and their limitations and future scope.
    Abstract

    Cloud computing had improved IT delivery in services like storage, computing, management and deployment with the help of automation and virtualization. So, security became an important factor in cloud computing. Cloud security testing is a new service model where organizations can perform testing on the cloud whenever required. This will help organizations to minimize the cost with maintaining cloud technologies and solutions in a secured way. However, there are some challenges in cloud security testing. Some of them are mentioned below. 1) Distributed risks with Distributed Computing: Distributed computing gives us leverage to use unlimited resource pool whenever required. This computing also comes with Distributed risks at the same time as clients will not be having access to internal operations of resources. Handling issues like Data segregation, leakage of private information, service loss and malware attacks are some of the risks with distributed computing. 2) On-Demand services: Client always expects cloud services to be available, reachable and should be able to integrate with other components all the time with data confidentiality. The service providers should offer support for tools integration. Moreover, clients also want some selective data to be exposed for testing. All this information need to be communicated to service providers. 3) Lack of Cloud security testing standards: There are no set of standards/methods for cloud security testing. It all depends on Client requirements. The approach which may be best suited for a client may not be useful for another. So, there is a need for quality of service and models which defines cloud security testing. So, Cloud based security testing make use of cloud resources for its testing whenever needed. Although there are some challenges in this domain, but still service providers are trying to enhance their security guidelines and protocols to provide the secured environment to the clients.
